VITAL MARK: BLOOD GROUP DETECTION USING FINGERPRINT IMAGE

Introduction

This research-based project aims to predict an individual's blood group using fingerprint images by leveraging deep learning techniques. Traditional blood group testing methods require blood samples, but this project explores a non-invasive approach using biometric analysis and artificial intelligence.

Description

The project involves capturing high-resolution fingerprint images and analyzing them using Convolutional Neural Networks (CNN). The system extracts unique fingerprint features and classifies them into different blood groups.
